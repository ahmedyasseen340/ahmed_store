# Aspire — ملفات الموقع

نسخة كاملة من شكل موقعك (Home / Catalog / Product / Cart / Contact) كـ HTML + CSS + JS عادي،
علشان تقدر تفتحها وتعدّل فيها زي ما انت عايز من غير أي منصة.

## هيكل الملفات

```
aspire-store/
├── index.html              ← الصفحة الرئيسية
├── catalog.html             ← صفحة كل المنتجات (فيها فلتر وترتيب)
├── css/
│   └── style.css            ← كل التصميم (الألوان، الخطوط، الفراغات) من هنا
├── js/
│   ├── products.js          ← بيانات المنتجات والريفيوز (ده أهم ملف لو هتغيّر المنتجات)
│   ├── cart.js               ← منطق السلة (الحفظ في المتصفح)
│   └── main.js                ← فتح/قفل القوائم، عرض الكروت، التوست
└── pages/
    ├── product.html          ← قالب صفحة المنتج (بيشتغل بأي منتج عن طريق ?slug=)
    ├── cart.html              ← صفحة السلة الكاملة
    ├── contact.html           ← صفحة تواصل معنا
    ├── privacy-policy.html
    ├── refund-policy.html
    └── shipping-policy.html
```

## أهم حاجة هتعدّلها: js/products.js

كل منتج في الموقع موجود كـ object في `PRODUCTS`. لو عايز تضيف منتج جديد، انسخ
أي block واحد وغيّر فيه:

- `slug` — الاسم اللي بيظهر في اللينك (لازم يكون فريد)
- `name`, `category`, `price`, `compareAt` (سعر قبل الخصم، أو `null` لو مفيش خصم)
- `inStock` — `true` أو `false`
- `sizes` — array بالمقاسات
- `images` — أول صورة وتاني صورة (تاني صورة بتظهر لما تحط الماوس على الكارت)
- `description` — وصف المنتج اللي بيظهر في صفحته

نفس الملف فيه `REVIEWS` لو عايز تغيّر آراء العملا اللي ظاهرة في الصفحة الرئيسية.

## لو عايز تجرب الموقع على جهازك

أبسط طريقة: افتح `index.html` مباشرة بالمتصفح. لو حسّيت إن أي حاجة مش شغالة
(زي تحميل المنتجات)، شغّل سيرفر محلي بسيط بدل ما تفتح الملف مباشرة:

```
cd aspire-store
python3 -m http.server 8000
```

وبعدين افتح `http://localhost:8000` في المتصفح.

## السلة (Cart)

السلة دلوقتي شغالة بالكامل من جوه المتصفح (localStorage) — تقدر تضيف منتجات،
تغيّر الكمية، تشيل منتج، وتشوف الإجمالي. لكن مفيش بوابة دفع حقيقية متوصلة
لسه. لما تيجي تربط الموقع بمزود دفع حقيقي (زي Paymob, Fawry, أو حتى Shopify
نفسه)، هتعدّل في زرار "Checkout" في `pages/cart.html` و `index.html`.

## الألوان والخطوط

كل حاجة خاصة بالتصميم (الألوان، الخط، المسافات) موجودة في أول ملف
`css/style.css` تحت `:root{ ... }` — غيّر القيم هناك وهتتغير في كل الموقع
مرة واحدة.
